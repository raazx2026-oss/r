# 🌙 Zikr & Social Community App

A comprehensive, premium-designed Android application built to bring the Ummah together. It offers daily Islamic motivation, a Zikr/Tasbeeh counter, step-by-step Tahajjud guides, expandable Quranic Surah folders, and a live Social Community feed. 

The app features a **Daily Jackpot Quiz** where users can win real UPI rewards, entirely manageable via a custom Firebase-powered Web Admin Panel.

---

## ✨ Key Features

### 📱 Android App (User Side)
* **Live Community Feed:** A social feed displaying Islamic posts, images, and HTML-formatted articles fetched in real-time.
* **Daily Jackpot Quiz:** A dynamic, multi-question quiz. Scoring 100% unlocks a secure Reward Claim page (UPI integration).
* **Quranic Surahs (Expandable Folders):** Read Surahs categorized by folders, complete with Arabic text, Roman transliteration, and Tarjuma (Translation).
* **Zikr & Tasbeeh Counter:** A digital counter with multiple built-in Zikr options and a reset function.
* **Prophets History & Motivation:** Daily updated stories of Prophets for inspiration.
* **Tahajjud Guide:** Step-by-step visual and text guide for night prayers.
* **State Persistence:** Remembers the last visited tab (e.g., Social Feed or Dashboard) for a seamless user experience.
* **Premium UI/UX:** Dark theme, gold accents, Material Design cards, and smooth slide-up animations.

### 💻 Master Admin Panel (Web)
* **100% Serverless Management:** Built with HTML, CSS, and Firebase JS SDK. No backend server required.
* **Live Content Updates:** Update the Daily Banner, Quiz Questions, Surahs, and Social Posts without pushing app updates to the Play Store.
* **Reward URL Manager:** Dynamically change the winning redirect URL for the daily quiz.
* **Mobile-Friendly UI:** The admin dashboard is fully responsive and can be operated directly from a smartphone.

---

## 🛠 Tech Stack

* **Frontend (Android):** Java, XML, Android SDK
* **UI Components:** Google Material Design (`MaterialCardView`, `NavigationView`, `DrawerLayout`)
* **Image Loading:** [Glide](https://github.com/bumptech/glide)
* **Backend:** Firebase Realtime Database
* **Admin Panel:** HTML5, CSS3, Vanilla JavaScript
